# CRUD Notas APP

### Dependencies:

- body-parser
- dotenv
- express
- mysql

### Quick Start

```bash
# Install dependencies
$ npm install
 
# Database
$ node_crud.sql

# Start App
$ node app/index.js

App Run On: http://localhost:3000

```

### Test the Notas APP

```bash
Server is running on port 3000.
Successfully connected to the database.
```
